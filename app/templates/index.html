Distributed Messaging Queue with RabbitMQ

A testing platform for RabbitMQ exchange types and MIME types in a distributed environment.

This application allows you to experiment with different RabbitMQ exchange types, message formats, and distributed messaging patterns. You can configure and test various scenarios to understand real-world trade-offs between reliability, scalability, and performance.

Exchange Types
Test different exchange types

Explore how different exchange types route messages between producers and consumers.

  • Direct Exchange: Route messages based on an exact routing key match
  • Fanout Exchange: Broadcast messages to all bound queues
  • Topic Exchange: Route messages based on wildcard pattern matching
  • Headers Exchange: Route messages based on header attributes
Try Exchange Examples
Message Formats & MIME Types
Test different message formats

Send and receive messages in various formats with different MIME types.

  • text/plain: Simple text messages
  • application/json: Structured JSON data
  • application/xml: XML formatted data
  • application/octet-stream: Binary data
